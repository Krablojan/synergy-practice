# Книжный магазин — Кейс-задача №4

## Функционал:

- Регистрация и вход (обычный пользователь и администратор)
- Два интерфейса: пользовательский и административный
- Каталог книг с сортировкой по категории, автору, году
- Аренда книги на 2 недели / 1 месяц / 3 месяца
- Админ-панель: добавление, редактирование, удаление книг, управление ценой и доступностью
- Автоматическая проверка сроков аренды и освобождение просроченных книг
- Все данные сохраняются в LocalStorage

## Технологии:
- Vue 3 (Composition API)
- Pinia (управление состоянием)
- Vue Router
- Чистый CSS

## Как запустить:

npm install
npm run dev
