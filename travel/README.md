# Дневник путешествий — Кейс-задача №5

### Реализованный функционал:
- Регистрация и вход пользователя
- Создание записи о путешествии (название, описание, стоимость, геопозиция, фото)
- Автоматическое определение местоположения (широта, долгота, адрес через OpenStreetMap)
- Загрузка и отображение изображения с превью
- Лента всех записей на главной странице
- Отдельная страница просмотра записи
- Все данные сохраняются в LocalStorage

### Технологии
- Vue 3 (Composition API)
- Pinia (управление состоянием)
- Vue Router
- Чистый CSS (современный дизайн с карточками, тенями, адаптивностью)
- Geolocation API + Nominatim (OpenStreetMap) для адреса

### Как запустить

npm install
npm run dev
